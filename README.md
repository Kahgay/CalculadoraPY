# CalculadoraPY

## Usando no Termux

```
apt install python3
apt install git
git clone https://github.com/Kahgay/CalculadoraPY.git
cd CalculadoraPY
python3 calc.py
```

## Usando no Linux

```
sudo apt-get install python3
sudo apt-get install git
sudo apt-get install python3-pip
sudo pip3 install termcolor
git clone https://github.com/Kahgay/CalculadoraPY.git
cd CalculadoraPY
python3 calc.py
```
