# CalculadoraPY

## Usando no termux

```
apt install python3
apt install git
git clone https://github.com/Kahgay/CalculadoraPY.git
cd CalculadoraPY
python3 calc.py
```

## Usando no Linux

```
apt-get install python3
apt-get install git
git clone https://github.com/Kahgay/CalculadoraPY.git
cd CalculadoraPY
python3 calc.py
```
